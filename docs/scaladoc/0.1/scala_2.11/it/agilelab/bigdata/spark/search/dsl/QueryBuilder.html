QueryBuilder

Related Doc: package dsl

case class QueryBuilder(field: field) extends Product with Serializable

The main entry point for the Query DSL.

The grammar for the DSL is as follows:

QUERY = all not QUERY QUERY && QUERY QUERY || QUERY

QUERY = FIELDNAME matchText "text" FIELDNAME matchPhrase "phrase" FIELDNAME matchTerm TERM FIELDNAME matchMin NUMBER from TERMSET FIELDNAME matchAll TERMSET FIELDNAME matchAny TERMSET FIELDNAME parseQuery "raw lucene query" FILTER

FILTER = FIELDNAME e FIELDVALUE FIELDNAME gt Num FIELDNAME gte Num FIELDNAME lt Num FIELDNAME lte Num FIELDNAME in RANGE FIELDNAME in VALUESET exists(FIELDNAME) missing(FIELDNAME)

TERM = term("term") FIELDNAME = field("field") FIELDVALUE = value("value")

TERMSET = termSet(TERM...) VALUESET = valueSet(FIELDVALUE..) RANGE = range(FIELDVALUE, FIELDVALUE)
